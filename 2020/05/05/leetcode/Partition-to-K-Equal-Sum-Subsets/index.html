Partition to K Equal Sum Subsets

Given an array of integers nums and a positive integer k, find whether it’s possible to divide this array into k non-empty subsets whose sums are all equal.

Example 1:

Note:

  • 1 <= k <= len(nums) <= 16.
  • 0 < nums[i] < 10000.

思路

解法一 递归+回溯

问题归结于将所有数均分为k分，每份的和为sum/k.可以先对数组进行一次过滤，如sum % k != 0 直接返回false，有单个值大于sum / k的情况也直接返回false。等等。最后的核心是尝试将当前值放到每个subset中，其中subset是均分后的每个小子集。如果能将所有的数放进去，则表示能均分，否则，不能。
